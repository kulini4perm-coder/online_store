# Online Store Project

Проект по разработке бэкенд-составляющей для интернет-магазина. Реализована базовая логика управления товарами и категориями.

## Стек технологий
- **Python 3.14**
- **Poetry** (управление зависимостями)
- **Pytest** (тестирование)

## Функционал
- Создание товаров с описанием, ценой и количеством.
- Группировка товаров в категории.
- Автоматический подсчет общего количества категорий и уникальных товаров в системе через атрибуты классов.

## Обновления 23.01.2026
**Класс Product**
- Инкапсуляция цены: Атрибут цены теперь является приватным.
- Проверка цены: Реализован сеттер, который предотвращает установку нулевой или отрицательной стоимости.
- Класс-метод: Создает объекты Product напрямую из словарей с данными.

**Класс Category**
- Безопасное хранение: Список товаров стал приватным.
- Метод add_product: Реализован единый интерфейс добавления товаров в категорию.
- Форматированный вывод: Геттер products возвращает список всех товаров в виде текстового отчета.

## Обновления 26.01.2026
**Строковое представление объектов (__str__):**
- Класс Product: теперь возвращает информацию в формате: Название, Цена руб. Остаток: X шт.
- Класс Category: выводит суммарное количество всех единиц товара в категории.

**Сложение товаров (__add__):**
- Реализована возможность складывать два объекта Product.
- Проверка типа: сложение доступно только между объектами класса Product.

**Тестирование:**
Добавлены тесты для проверки корректности математических вычислений при сложении товаров.
Реализованы тесты на проверку строкового вывода категорий и продуктов.
Все тесты используют общие фикстуры из conftest.py и поддерживают строгую аннотацию типов.

## Обновления 28.01.2026
